**My second mini-project** — Built during 2nd semester at **P.A. College of Engineering, Mangalore**.

An IoT-based smart device to help users take medications on time with timely reminders and remote monitoring.

## 🎯 Project Overview

This project helps solve the common problem of forgetting medication by providing scheduled alerts and allowing family members to monitor the status remotely.

## ✨ Features

- Scheduled medicine reminders using RTC
- Buzzer alerts for timely notifications
- IR sensor to detect if medicine was taken
- Real-time status monitoring via ThingSpeak cloud
- Wi-Fi connectivity with ESP8266

## 🛠️ Technologies & Components Used

- **Microcontroller**: ESP8266
- **Timing Module**: RTC (DS3231)
- **Sensor**: IR Sensor
- **Output**: Buzzer
- **Cloud Platform**: ThingSpeak
- **Development Environment**: Arduino IDE
- **Programming Language**: C/C++

## 📌 How It Works

The RTC keeps track of medicine timings. At scheduled times, the buzzer alerts the user. The IR sensor detects if the medicine box was opened/taken. All status data is sent to ThingSpeak for remote monitoring.
